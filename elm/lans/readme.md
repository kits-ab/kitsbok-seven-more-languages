# Elm v.0.16.0

## Install instructions
1. Install latest version of node 
2. Install v0.16.0 of elm
```sh
npm i -g elm@0.16.0
```
3. cd to folder elm_0.16.0
4. Start reactor
```sh
elm reactor
```
5. Open http://localhost:8000
